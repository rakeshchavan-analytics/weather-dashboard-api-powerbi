# 🌦️ Live Weather Dashboard

## 📌 Project Overview
This project is a **real-time weather dashboard** built using **Power BI** by integrating live data from a Weather API.  
It provides insights into current weather conditions, forecasts, and air quality in an interactive and visually appealing format.

---

## 🚀 Key Features
- 🌡️ Real-time temperature updates  
- 📅 7-day weather forecast  
- 🌅 Sunrise & Sunset timings  
- 🌧️ Chance of rain analysis  
- 🌬️ Wind speed, humidity, pressure tracking  
- 🌫️ Air Quality Index (AQI) with pollutant breakdown  
- 📊 Clean and modern dashboard design

---

## ⚙️ How It Works
1. Fetch real-time data using Weather API  
2. Transform data using Power Query  
3. Build data model in Power BI  
4. Create interactive visuals and insights
